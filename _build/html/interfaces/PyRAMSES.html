3.3. Python InterfaceÂ¶

PyRAMSES is Python module that facilitates the use of the dynamic simulator RAMSES. The module provides basic functionality like defining test-cases, running simulation, extracting information, and many more.

The module includes inside pre-compiled versions of RAMSES in the form of dynamic libraries. It supports both Windows and Linux OSs. You can use the sidebar on the left to navigate through the documentation for more information on installing, updating, and using PyRAMSES.
